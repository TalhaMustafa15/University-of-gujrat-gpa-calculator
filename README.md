# University-of-gujrat-gpa-calculator
This code is a complete **GPA Calculator web application** built using **HTML, CSS, and JavaScript** with a simple user authentication system.
First, it provides **Login and Signup functionality** where users can create an account and log in. The data (users and sessions) is stored in the browser using **localStorage**, so no backend is required. It also includes basic validation like name length, email format, and password length.
After login, the user accesses the **main GPA calculator page**. Here, they can add courses by entering the course name, grade, and credit hours. The system converts grades into grade points, calculates total credits and GPA, and displays the result dynamically in a table. Users can also remove courses and log out easily.
Overall, this is a **frontend-based student tool** that combines authentication and GPA calculation in a single responsive webpage.
